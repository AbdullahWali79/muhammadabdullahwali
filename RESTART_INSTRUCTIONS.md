# React App Restart Instructions

## Step 1: Stop Current App (Agar chal raha hai)

1. Terminal/Command Prompt mein jahan app chal rahi hai
2. `Ctrl + C` press karo (app stop ho jayegi)

## Step 2: Clear Cache (Optional but recommended)

```bash
# npm cache clear (agar zarurat ho)
npm start
```

## Step 3: Start App Again

```bash
npm start
```

## Step 4: Browser mein Hard Refresh

1. Browser mein app kholo
2. `Ctrl + Shift + R` press karo (Windows/Linux)
   ya
   `Cmd + Shift + R` press karo (Mac)

Yeh purani cached files clear karke nayi configuration load karega.

---

**Important:** Naya Supabase project configure karne ke baad app restart karna zaroori hai taake nayi credentials load ho sakein!
